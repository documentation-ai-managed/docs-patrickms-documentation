## API Integrations

Connect your websites with third-party services and platforms through our extensive API.

<CodeGroup tabs="JavaScript,PHP">
  ```javascript
  const novagestion = require('novagestion-api');
  
  const client = new novagestion.Client({
    apiKey: 'YOUR_API_KEY'
  });
  
  client.websites.list().then(websites => {
    console.log('Websites:', websites);
  });
  ```
  ```php
  <?php
  require 'vendor/autoload.php';
  
  use NovaGestion\API\Client;
  
  $client = new Client('YOUR_API_KEY');
  $websites = $client->websites()->list();
  echo 'Websites: ' . json_encode($websites);
  ?>
  ```
</CodeGroup>

<Callout kind="info">Our SDKs support multiple programming languages for seamless integration.</Callout>
